# Restaurant-Chain-Mockup
## 🎯 学習の目的
1. **PHP オブジェクト指向 (OOP) の実践**
   * クラス、継承、カプセル化などの概念を実際のコードに落とし込む。
   * 拡張性が高く、メンテナンスしやすいコード設計を学ぶ。
2. **動的サーバーの構築練習**
   * Linux環境での環境構築（Nginx, PHPのセットアップ）。
   * クライアントからのリクエストに対して動的にレスポンスを返す仕組みの理解。

## 🛠 技術スタック
* **Language:** PHP 8.x
* **Infrastructure:** AWS EC2 (Ubuntu / Linux)
* **Web Server:** Nginx
* **Tools:** VS Code, Git/GitHub
